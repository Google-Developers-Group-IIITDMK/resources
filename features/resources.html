    Resources

    Here are curated resources to help you learn and grow:

    How to use this page

    Bookmark this page and check back for updated resources.
    Each section includes trusted links with a short description.

    Programming

    Programming Resources

    • GeeksforGeeks — Tutorials, articles, and coding problems on data structures, algorithms, and more.
    • LeetCode — Practice coding challenges, mock interviews, and contests for coding interviews.
    • HackerRank — Platform for coding practice, contests, and interview preparation.
    • Codeforces — Competitive programming contests and practice problems.
    • CodeChef — Coding challenges, long contests, and a supportive programming community.
    • freeCodeCamp — Free interactive lessons and projects covering web dev, algorithms, and data science.
    • Codecademy — Interactive courses for programming languages, web dev, and data science.

    Cloud

    Cloud Platforms

    • Google Cloud Skills Boost — Hands-on labs and learning paths for Google Cloud.
    • Qwiklabs — Temporary cloud environments for practice with Google Cloud and AWS.
    • AWS Training & Certification — Official AWS training courses and certification preparation.
    • A Cloud Guru — Specialized platform for AWS, Azure, and Google Cloud learning paths.
    • Pluralsight — Large collection of cloud, DevOps, and IT training courses.

    Mobile & Web

    Frontend

    Frontend Development

    • React — Library for building user interfaces and modern web apps.
    • MDN Web Docs — In-depth documentation and tutorials on HTML, CSS, and JavaScript.
    • freeCodeCamp Web Dev Curriculum — Beginner-friendly projects and lessons on frontend development.

    Backend

    Backend Development

    • Node.js — JavaScript runtime for building scalable server-side apps.
    • Django — Python-based web framework for rapid backend development.
    • Express.js — Fast and minimalist web framework for Node.js.
    • Spring Boot — Java framework for backend APIs and microservices.

    Cross-platform

    Cross-platform

    • Flutter — Framework for building natively compiled mobile & web apps from a single codebase.

    Machine Learning & AI

    ML & AI Resources

    • TensorFlow — Tutorials and guides for building machine learning models.
    • Kaggle — Datasets, competitions, and free machine learning courses.
    • Coursera: Machine Learning Introduction — Beginner-friendly specialization to understand ML theory and practice.
    • fast.ai — Practical deep learning courses designed for coders.
    • DeepLearning.AI — Specializations and courses on deep learning, NLP, and AI applications.

    DevOps

    DevOps Tools & Learning

    • Docker — Official docs and tutorials on containerization.
    • Kubernetes — Official docs for container orchestration and cluster management.
    • Terraform — Infrastructure as code (IaC) documentation and tutorials.
    • Jenkins — Automation server for CI/CD pipelines.
    • GitHub Actions — Automate workflows directly within GitHub repositories.
    • Linux Foundation Training — DevOps, cloud-native, and Linux system administration courses.

    Cryptography

    Cryptography Resources

    • Beginner-friendly introduction to practical modern cryptography. Crypto101
    • Hands-on cryptography challenges that teach real-world attack/defense techniques. Cryptopals
    • Stanford / Dan Boneh — an excellent introductory Coursera course on cryptographic principles. Cryptography I — Coursera
    • Authoritative essays and analysis on cryptography and security by Bruce Schneier. Schneier on Security (Cryptography)
    • Browser API reference and guides for performing cryptography in web applications. MDN — Web Crypto API
    • Modern, easy-to-use crypto library and docs (recommended for secure primitives). libsodium Documentation
    • Python cryptography library withrecipes and secure primitives for developers. PyCA Cryptography
    • Official standards, recommendations, and guidance for cryptographic algorithms (NIST). NIST Cryptographic Toolkit
    • Q&A site focused on cryptography research and engineering questions. Cryptography Stack Exchange
    • Source of RFCs and protocol specifications related to cryptography and internet standards. RFC Editor
